2) NAT

Koja je izvorišna, a koja odredišna IP adresa svakog ICMP echo request paketa snimljenog u programu Wireshark? Kojim čvorovima pripadaju te IP adrese?

Pingao sam cvor mm u Internet kartici, sa ip adresom 20.0.0.4.
Izvorna IP adresa je 192.168.0.20 je od pc-a1 a odredistna 20.0.0.4.

Istražite ove dvije naredbe. Što radi svaka od njih? 

Naredbe sluze za konfiguraciju NAT tablica.
    iptables -t nat -A POSTROUTING --out-interface <vanjsko sučelje> -j MASQUERADE       
	
	-j MASQUERADE --> izvorisna IP adresa se mijenja adresom izlaznog interfaca    
	--out-interface eth2 --> izlazni interface
	-t nat --> definicija tablice prilikom dolaska paketa
	-A POSTROUTING --> izmjena paketa prilikom izlaza iz usmjeritelja	
	
	iptables -A FORWARD --in-interface <unutarnje sučelje> -j ACCEPT
	
	-A FORWARD --> prosljeduje pakete
	--in-interface --> ulazni interface
	-j ACCEPT --> prihvaca pakete	
    
	
	iptables -A FORWARD --in-interface eth1 -j ACCEPT
	iptables -t nat -A POSTROUTING --out-interface eth2 -j MASQUERADE
